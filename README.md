# 大地のムクナ LP — 確定融合版

## 構成
1. 固定ヘッダー
2. C案準拠の一体型ファーストビュー
3. 小型の共感ブリッジ
4. 発酵 × ムクナ豆 × 糸島の自然
5. 毎日続けたくなる三つの理由
6. 先行販売のお声
7. LINE登録で受け取れる特典
8. 最終CTA
9. 中央揃えフッター

## ローカル確認
```powershell
npm.cmd install
npm.cmd run dev
```
ブラウザで `http://localhost:3000` を開きます。

## ビルド
```powershell
npm.cmd run build
```

## 重要
- `lib/site.ts` のLINE URLを公開前に公式URLと照合してください。
- コンペ終了後は `IS_COMPETITION_PREVIEW` をfalseにするか公開を停止してください。
- 表示根拠は `CLIENT_CONFIRMATION_CHECKLIST.md` を確認してください。
